# Road Map Pessoal — Dev Front-end

Roadmap interativo de estudos para desenvolvimento front-end, organizado em fases progressivas com acompanhamento de progresso.

## Fases de Estudo

| Fase | Conteúdo |
|------|----------|
| 01 | JavaScript de verdade |
| 02 | JavaScript Assíncrono |
| 03 | React do Zero |
| 04 | Preparação pro Emprego |

## Como usar

Abra o arquivo `roadmap_frontend.html` diretamente no navegador.

- Marque os itens conforme for concluindo
- A barra de progresso global é atualizada automaticamente
- Cada fase pode ser expandida/recolhida clicando no cabeçalho

## Tecnologias

- HTML, CSS e JavaScript puro
- Fontes: [Space Mono](https://fonts.google.com/specimen/Space+Mono) e [Syne](https://fonts.google.com/specimen/Syne) (Google Fonts)
